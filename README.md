# fusion360-NoGestures
Disable gestures and make the right mouse button act as the middle one

[Support / discussion](https://forums.autodesk.com/t5/fusion-360-api-and-scripts/nogestures-add-on-from-pawel-inc-support/m-p/7757972) thread on Autodesk's forum.

[Link](https://apps.autodesk.com/FUSION/en/Detail/Index?id=1331850769524044714) to download the addon.

For now only Windows is supported.
